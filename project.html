Project Overview

This ongoing project focuses on the exploration and development of innovative textile-based interfaces for automotive interiors. The goal is to create a 3-dimensional, touch-sensitive cover for the middle console that integrates seamlessly into the car's design and enhances user interaction.

Key Features

  • Explores the affordances of textile surfaces for tactile interaction.
  • Integration of textiles in 3D surfaces for an intuitive multi-surface interaction.
  • Focus on enhancing the user experience within automotive environments.

Status

The project is currently in the prototyping phase, with ongoing testing and refinement of the interface design.
